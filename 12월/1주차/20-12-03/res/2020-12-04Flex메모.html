flex box 자식요소의 속성 -flex container 의 직계 자손요소는 자동적으로 flex 요소가 됨 -부모 태그의 CSS 속성에 display:flex를 사용하면 직계 자손 태그는 자동적으로 flex 요소가 됨 -order:flex 요소의 순서를 설정하는 속성 -flex-grow:flex 요소의 크기를 설정하는 속성, 다른 요소의 크기보다 몇배로 커질지를 설정, 기본값 0 -flex-shrink:flex 요소의 크기를 설정하는 속성, 다른 요소의 크기보다 얼마나 작아질 것인지를 설정, 기본값 1 -flex-basis:flex 요소의 초기 크기를 설정 -flex:order,flex-grow,flex-shrink의 속성을 한번에 설정하는 속성
